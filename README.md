# Push a Box #
Game de plataforma 2D.

### **Objetivo do Game** ###
- Fazer o personagem em 2d empurrar uma caixa do inicio até o final da plaforma e mostrar uma mensagem de vitoria na tela.

### **Tecnologias** ###
* A aplicação  em C#.

### **Configurações** ###
* Aspecto 16:9

### **Controles** ###
* Shift Esq + Seta: Correr
* Seta Dir: Direita
* Seta Esq: Esquerda
* Espaço: Pular
